Trang mẫu — ID Selector

Ví dụ minh hoạ cách sử dụng id selector trong CSS

Giải thích nhanh

ID selector dùng dấu # theo sau tên id, ví dụ #main — áp dụng cho phần tử có attribute id="main".

Specificity: id selector có specificity cao hơn element selector và class selector, vì vậy dễ dùng để ghi đè style mặc định.

Ví dụ nút

Ghi chú: Mỗi id nên duy nhất trong một trang HTML.

Side panel (id)

Panel này dùng id side-panel để style riêng.
