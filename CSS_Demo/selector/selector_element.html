Element selector là gì?

Element selector chọn phần tử theo tên thẻ HTML. Ví dụ: p { color: red; } sẽ áp dụng cho tất cả thẻ <p>.

Element selector rất hữu dụng để thiết lập kiểu mặc định cho cấu trúc văn bản (heading, paragraph, links,...).
