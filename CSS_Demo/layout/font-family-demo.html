Demo: CSS Font — font-family

File: font-family-demo.html

1) Serif
Đây là ví dụ sử dụng font serif: Times New Roman.
.serif { font-family: "Times New Roman", Times, serif; }
2) Sans-serif
Đây là ví dụ với font sans-serif: Arial.
.sansserif { font-family: Arial, Helvetica, sans-serif; }
3) Monospace
Đây là ví dụ với font monospace: Courier New.
.monospace { font-family: "Courier New", Courier, monospace; }
4) Cursive
Đây là ví dụ với font cursive: Brush Script MT.
.cursive { font-family: "Brush Script MT", cursive; }
5) Fantasy
Đây là ví dụ với font fantasy: Papyrus.
.fantasy { font-family: Papyrus, fantasy; }
6) Google Font
Đây là ví dụ với font Google Roboto.
.googlefont { font-family: "Roboto", sans-serif; }

Ghi chú nhanh

  • Các họ chính: serif, sans-serif, monospace, cursive, fantasy.
  • Luôn khai báo danh sách fallback fonts (ví dụ: Arial, Helvetica, sans-serif).
  • Google Fonts có thể import qua thẻ <link> hoặc @import trong CSS.
