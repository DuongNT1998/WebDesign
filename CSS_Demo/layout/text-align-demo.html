Demo: CSS Text — text-align

File: text-align-demo.html

1) text-align: left (mặc định)
Văn bản sẽ căn lề trái mặc định.
Dòng thứ hai cũng căn trái.
.left { text-align: left; }
2) text-align: right
Văn bản được căn sang phải.
Dùng khi muốn số hoặc ngày tháng thẳng hàng bên phải.
.right { text-align: right; }
3) text-align: center
Văn bản căn giữa.
Thường dùng cho tiêu đề.
.center { text-align: center; }
4) text-align: justify
Văn bản được giãn ra để canh đều hai bên trái và phải, tạo cảm giác gọn gàng hơn trong đoạn văn bản dài. Tuy nhiên, có thể xuất hiện khoảng trắng không đều giữa các từ.
.justify { text-align: justify; }
5) text-align: start

Với ngôn ngữ từ trái sang phải (LTR) → căn trái.

مع اللغة من اليمين إلى اليسار (RTL) → căn phải.

.start { text-align: start; }
6) text-align: end

Với ngôn ngữ từ trái sang phải (LTR) → căn phải.

مع اللغة من اليمين إلى اليسار (RTL) → căn trái.

.end { text-align: end; }

Ghi chú nhanh

  • left, right, center, justify là 4 giá trị quen thuộc nhất.
  • start và end linh hoạt theo chiều viết của ngôn ngữ (LTR hoặc RTL).
  • Để canh giữa theo chiều dọc, không dùng text-align mà dùng Flexbox hoặc Grid.
