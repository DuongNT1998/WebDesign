Demo: CSS Text — text-indent

File: text-indent-demo.html

1) text-indent: 30px
Đây là một đoạn văn có thụt đầu dòng 30px. Thuộc tính text-indent được áp dụng cho dòng đầu tiên của đoạn văn.
.indent-30px { text-indent: 30px; }
2) text-indent: 10%
Đoạn văn này thụt đầu dòng theo tỉ lệ 10% chiều rộng của phần tử cha. Nếu khung rộng, khoảng thụt cũng lớn hơn.
.indent-10p { text-indent: 10%; }
3) Giá trị âm (negative)
Dòng đầu tiên bị kéo ngược vào bên trái với giá trị -2em. Thường dùng để tạo kiểu danh sách đặc biệt hoặc chú thích.
.negative { text-indent: -2em; }
4) Hanging indent (thụt dòng treo)
Kỹ thuật "hanging indent": dòng đầu tiên nhô ra ngoài, còn các dòng tiếp theo thì thụt vào. Ứng dụng trong tài liệu tham khảo, thư mục (bibliography).
.hanging {
  text-indent: -2em;
  padding-left: 2em;
}

Ghi chú nhanh

  • text-indent chỉ áp dụng cho dòng đầu tiên của đoạn văn bản.
  • Có thể dùng đơn vị tuyệt đối (px, em) hoặc tương đối (%).
  • Giá trị âm thường dùng để tạo hiệu ứng đặc biệt (ví dụ hanging indent).
