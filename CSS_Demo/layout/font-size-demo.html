Demo: CSS Font — font-size

File: font-size-demo.html

1) Đơn vị tuyệt đối (px)
Văn bản này có font-size: 16px (kích thước cố định).
.px16 { font-size: 16px; }
2) Đơn vị tương đối (em)
font-size: 2em (gấp đôi kích thước font của phần tử cha).
.em2 { font-size: 2em; }
3) Đơn vị tương đối (%)
font-size: 150% (bằng 1.5 lần kích thước font của phần tử cha).
.percent150 { font-size: 150%; }
4) rem (root em)
font-size: 2rem (gấp đôi font-size của <html>, không phụ thuộc cha).
.rem2 { font-size: 2rem; }
5) Từ khóa tuyệt đối
font-size: xx-small (một trong các keyword: xx-small → xx-large).
.xxsmall { font-size: xx-small; }
6) Từ khóa tương đối
font-size: larger (tăng một cấp so với phần tử cha).
.larger { font-size: larger; }

Ghi chú nhanh

  • px cho kích thước cố định; em, % cho tỉ lệ theo phần tử cha.
  • rem hữu ích để tạo kích thước font nhất quán theo root.
  • smaller, larger thay đổi so với phần tử cha theo cấp.
  • Các keyword tuyệt đối: xx-small, x-small, small, medium, large, x-large, xx-large.
