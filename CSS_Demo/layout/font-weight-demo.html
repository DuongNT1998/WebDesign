Demo: CSS Font — font-weight

File: font-weight-demo.html

1) font-weight: 300 (Light)
Văn bản với font-weight: 300 (mảnh, nhẹ).
.light { font-weight: 300; }
2) font-weight: 400 (Normal)
Văn bản với font-weight: 400 (mặc định).
.normal { font-weight: 400; }
3) font-weight: 500 (Medium)
Văn bản với font-weight: 500 (đậm vừa).
.medium { font-weight: 500; }
4) font-weight: 700 (Bold)
Văn bản với font-weight: 700 (in đậm).
.bold { font-weight: 700; }
5) font-weight: bolder
Văn bản với font-weight: bolder (đậm hơn phần tử cha).
.bolder { font-weight: bolder; }
6) font-weight: lighter
Văn bản với font-weight: lighter (mảnh hơn phần tử cha).
.lighter { font-weight: lighter; }

Ghi chú nhanh

  • Giá trị số thường dùng: 100 → 900 (100 = rất mảnh, 900 = rất đậm).
  • normal = 400, bold = 700.
  • bolder, lighter phụ thuộc vào font cha.
  • Không phải font nào cũng hỗ trợ đủ các mức (thường chỉ có 400 và 700).
