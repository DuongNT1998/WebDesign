Demo: CSS Font — font-style

File: font-style-demo.html

1) font-style: normal
Văn bản này ở kiểu chữ bình thường (normal).
.normal { font-style: normal; }
2) font-style: italic
Văn bản này sử dụng kiểu italic (in nghiêng).
.italic { font-style: italic; }
3) font-style: oblique
Văn bản này sử dụng kiểu oblique (nghiêng giả).
.oblique { font-style: oblique; }

Ghi chú nhanh

  • normal: văn bản giữ nguyên kiểu chữ mặc định.
  • italic: chữ nghiêng thật (dựa vào font có sẵn kiểu italic).
  • oblique: chữ nghiêng giả (trình duyệt tự tạo bằng cách làm nghiêng chữ).
