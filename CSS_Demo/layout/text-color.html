Demo: CSS Text - Color

File: text-color-demo.html

1) Named color

color: tomato; — tên màu có thể dùng trực tiếp (ví dụ: black, white, red, tomato...)

p.named { color: tomato; }
2) Hexadecimal

color: #ff8800; — mã hex 6 chữ số.

color: #ff8800cc; — hex 8 chữ số (alpha ở cuối).

p.hex { color: #ff8800; }
3) rgb() / rgba()

color: rgb(34,139,230); — không có alpha.

color: rgba(34,139,230,0.6); — có alpha (độ mờ).

p.rgba { color: rgba(34,139,230,0.6); }
4) hsl() / hsla()

color: hsl(200 80% 50%); — Hue Saturation Lightness.

color: hsl(200 80% 50% / 0.55); — alpha bằng dấu gạch chéo.

p.hsla { color: hsl(200 80% 50% / 0.55); }
5) CSS Variables

color: var(--brand); — dùng biến để tái sử dụng màu.

color: var(--accent);

:root{ --brand: #0066cc; }
6) currentColor & inherit
Parent color = mediumseagreen
child: color: currentColor;
child: color: inherit;
.child { color: currentColor; /* dùng màu hiện tại của element */ }
7) Transparent / invisible text

color: transparent; — chữ "ẩn" nhưng vẫn có thể chọn (select) được.

p.transparent { color: transparent; }
8) Text gradient (sáng tạo)

Đoạn text gradient sử dụng background-clip: text.

.gradient-text{ background-clip:text; color:transparent; }
9) Contrast examples
Dark text
color:#111
Light text
color:#fff
Lưu ý: luôn kiểm tra độ tương phản (WCAG) nếu cần truy cập cho mọi người.

Ghi chú nhanh

  • Ưu tiên dùng biến CSS (var(--name)) để dễ bảo trì màu toàn bộ trang.
  • Sử dụng alpha (rgba / hsla / hex8) khi cần tạo hiệu ứng mờ/translucent.
  • currentColor rất tiện khi muốn các phần tử con dùng cùng màu với phần tử cha (ví dụ icon SVG).
  • Text gradient dùng background-clip: text và color: transparent (cần prefix cho trình duyệt cũ: -webkit-background-clip).
  • Kiểm tra tương phản màu để đảm bảo độ đọc cho người dùng (ví dụ: văn bản nhỏ trên nền sáng/tối).
