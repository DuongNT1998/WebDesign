Demo: CSS Text — text-decoration

File: text-decoration-demo.html

1) underline
Văn bản có gạch chân.
.underline { text-decoration: underline; }
2) overline
Văn bản có gạch ở phía trên.
.overline { text-decoration: overline; }
3) line-through
Văn bản có gạch ngang giữa.
.line-through { text-decoration: line-through; }
4) none
Văn bản không có trang trí nào. (Hữu ích để bỏ gạch chân mặc định của thẻ <a>)
.none { text-decoration: none; }
5) Kết hợp nhiều dòng
Văn bản vừa gạch trên vừa gạch dưới.
.combo { text-decoration: underline overline; }
6) Kiểu gạch chân
underline dotted (chấm).
underline double (hai vạch).
underline wavy đỏ (lượn sóng).
.wavy { text-decoration: underline wavy red; }
7) Màu sắc & độ dày
Gạch chân màu cam, dày 3px.
.custom {
  text-decoration-line: underline;
  text-decoration-color: #ff8800;
  text-decoration-thickness: 3px;
}

Ghi chú nhanh

  • text-decoration có thể gồm: line (underline, overline, line-through, none), style (solid, dotted, double, wavy...), color, thickness.
  • Dùng text-decoration: none; thường để bỏ gạch chân mặc định của liên kết.
  • Thay vì <u>, hãy dùng CSS để kiểm soát gạch chân.
