Ký tự đặc biệt trong HTML

Các ký tự đặc biệt được viết bằng HTML Entities, bắt đầu bằng & và kết thúc bằng ;.

1. Ký tự cơ bản

Dấu nhỏ hơn: < (gõ &lt;)

Dấu lớn hơn: > (gõ &gt;)

Dấu và: & (gõ &amp;)

Dấu nháy kép: " (gõ &quot;)

Dấu nháy đơn: ' (gõ &apos;)

2. Ký hiệu toán học

+ : &plus;

− : &minus;

× : &times;

÷ : &divide;

≤ : &le;

≥ : &ge;

3. Ký hiệu tiền tệ

$ : &dollar;

€ : &euro;

£ : &pound;

¥ : &yen;

₫ : &#8363; (Ký hiệu VNĐ)

4. Ký hiệu mũi tên

← : &larr;

→ : &rarr;

↑ : &uarr;

↓ : &darr;

5. Một số ký hiệu khác

© 2025 : &copy;

® : &reg;

™ : &trade;

♥ : &hearts;

★ : &starf;
