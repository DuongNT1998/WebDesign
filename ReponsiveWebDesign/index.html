Thiết kế Web Đáp ứng — Ví dụ dễ hiểu

Trang này minh họa cách dùng các độ đo CSS (px, rem, em, %, vw, vh) kết hợp với media queries và ảnh responsive (srcset/ sizes) để tạo giao diện thích ứng với mọi kích thước màn hình.

Bắt đầu
Tip: thay đổi kích thước cửa sổ trình duyệt để thấy hiệu ứng responsive.

Ví dụ các độ đo CSS

px (pixels)

Kích thước cố định — phù hợp với tài sản đồ họa, border, icon.

Ví dụ: width: 320px; height: 48px;

rem

Tương đối với font-size gốc (root). Dễ quản lý scale toàn cục.

Ví dụ: padding: 1.5rem;

% / vw / vh

Phù hợp layout linh hoạt: % phụ thuộc parent, vw/vh phụ thuộc viewport.

Ví dụ: width:90%; height:50vh;

Gallery (Grid responsive)

Grid tự chia cột theo breakpoint, ảnh dùng object-fit để cover.

Ghi chú & mẹo

  • Ưu tiên fluid layout (%, vw) cho container chính, kết hợp max-width để giới hạn chiều rộng trên màn hình lớn.
  • Sử dụng clamp() cho font-size để scale mượt giữa các breakpoint (đã dùng trong ví dụ: clamp(1.6rem, 2.5vw, 2.8rem)).
  • Ảnh responsive: luôn đặt alt và dùng srcset + sizes để trình duyệt chọn file phù hợp.
  • Kiểm tra mọi thứ trên thiết bị thật (mobile/tablet) vì emulator không phải lúc nào cũng chính xác.
